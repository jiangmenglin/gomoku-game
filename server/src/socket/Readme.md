# 这是一个由Claude 4.6 Opus一次性自动生成的游戏

## 运行指南

### 前提条件
- **Node.js** 已安装（v16+）
- **MongoDB** 已在本地运行（端口 27017）

### 启动步骤

```bash
# 1. 启动 MongoDB（确保已运行）

# 2. 启动后端服务
cd gomoku-game/server
npm install
npm run dev

# 3. 新开终端，启动前端
cd gomoku-game/client
npm install
npm start

# 4. 浏览器访问 http://localhost:3000
```

### 测试方法

1. 打开 **两个浏览器标签页**（或一个普通窗口 + 一个隐身窗口）
2. 分别注册两个账号并登录
3. 两个账号都点击 **"开始匹配"**
4. 匹配成功后即可开始对战！

### 功能总览

| 功能 | 说明 |
|------|------|
| ✅ 注册/登录/退出 | JWT 认证，密码 bcrypt 加密 |
| ✅ 在线玩家列表 | 实时显示状态（空闲/游戏中/离开） |
| ✅ 自动匹配 | 点击开始匹配，自动配对对手 |
| ✅ 五子棋对战 | 15×15 棋盘，轮流下子，实时同步 |
| ✅胜负/平局判定 | 五连珠判胜，225步满棋盘判平局 |
| ✅ 战绩记录 | 胜/负/平局自动记录到用户信息 |
| ✅ 好友系统 | 游戏中可加好友，已是好友显示标识 |
| ✅ 游戏结束返回大厅 | 自动解散匹配，回到房间页面 |
| ✅ 断线处理 | 对手断线自动判负 |
